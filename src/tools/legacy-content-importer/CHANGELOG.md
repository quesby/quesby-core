# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-01-20

### Added
- Initial release
- Support for importing legacy markdown content to Neutrino ULID system
- Command-line interface with multiple options
- Field mapping for common frontmatter formats
- Backup functionality
- Dry-run mode for testing
- Verbose output option
- Support for custom author and image paths
- Automatic alias generation for old URLs
- ULID generation for content identification
